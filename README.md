# TMF Intelligence System
### Alimentiv GI Clinical Trials — Powered by Claude API

---

## What This Is

A working AI prototype that demonstrates what Claude enablement looks like 
for a CRO TMF team. Built using real public GI trial protocols from 
ClinicalTrials.gov — all in Alimentiv's core IBD/GI domain.

**6 Modules:**
1. Study Overview — search and drill into any trial
2. TMF Tracker — document completeness per study
3. Flags Dashboard — critical issues across all studies
4. Query Studies — ask Claude anything across all protocols
5. Draft Communications — auto-generate site follow-up emails
6. Executive Summary — one-click audit-ready summaries

---

## Setup — Do This Once

### 1. Install Python dependencies
```bash
pip install -r requirements.txt
```

### 2. Set your Anthropic API key
```bash
# Mac / Linux
export ANTHROPIC_API_KEY=your_api_key_here

# Windows
set ANTHROPIC_API_KEY=your_api_key_here
```

### 3. Run the one-time data setup script
```bash
python setup_data.py
```

This will:
- Download 5 real GI trial PDFs from ClinicalTrials.gov
- Extract and chunk all text
- Create searchable index
- Save everything to data/

Takes about 5-10 minutes. Only needs to run once.

---

## Run The App

```bash
streamlit run app.py
```

Opens in your browser at http://localhost:8501

---

## Deploy to Streamlit Cloud (Share a Live URL)

1. Push this folder to a GitHub repository
2. Go to share.streamlit.io
3. Connect your GitHub repo
4. Add ANTHROPIC_API_KEY in the Secrets settings
5. Deploy — get a shareable URL instantly

## Architecture

```
ClinicalTrials.gov PDFs
        ↓
setup_data.py (run once)
        ↓
data/trial_index.json (searchable chunks)
        ↓
src/query_engine.py (search + Claude API)
        ↓
app.py (Streamlit dashboard)
        ↓
User browser
```

**Production path:**
- PDFs → SharePoint / Azure Blob Storage
- Index → Azure AI Search (vector database)
- App → Azure App Service
- Same Claude API, same logic
